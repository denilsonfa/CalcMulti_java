### SOBRE
###### Para a criação de um layout, aplicando o conceito do The 8-Point Grid, deve-se ter controle sobre números múltiplos, preferencialmente de 8, para que seja possível criar um melhor design com um bom espaçamento e uma boa harmonização. Entretanto, para que que se possa ter ganho de tempo, tive a ideia de criar um simples calculador em Java, para que eu possa visualizar uma pequena tabuada de números entre números previamente definidos.
###### Nesta calculadora recebe-se o número mínimo, número máximo e os intervalos, onde, através de uma simples condição, onde o resto da multiplicação entre um dos números dentro dos mínimos e máximos pelo número múltiplo resulte em 0 será registrado este, em uma string.
Este projeto está sujeito a futuras melhorias, entregando, no momento, o necessário para que a aplicação nos trabalhos de design.
